## Getting Started

-   Git clone the project
-   Install MeteorJS: https://docs.meteor.com/install.html
-   `meteor npm i`
-   `npm run dev`

### Updating Packages

-   Meteor: `meteor update`
-   NPM: `npx npm-check-updates -u` and then `npm install`
